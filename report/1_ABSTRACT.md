# ABSTRACT

This report presents the design, development, and evaluation of a dual-mode vision-guided robotic arm capable of both autonomous sorting and human-in-the-loop manipulation. The system addresses the limitations of traditional industrial manipulators by integrating advanced computer vision, real-time gesture recognition, and reliable motion planning within the Robot Operating System 2 (ROS 2) ecosystem. The proposed architecture employs a modular framework, switching dynamically between an autonomous vision-driven mode utilizing YOLOv8 for object detection, and a teleoperation mode utilizing MediaPipe for gesture-based human-in-the-loop control. 

Motion planning and kinematics are handled by MoveIt2, seamlessly translating high-level commands into actionable trajectories. Control execution is achieved via a customized hardware interface employing an ESP32 microcontroller and a PCA9685 PWM driver, bridging the gap between high-level ROS 2 nodes and low-level servomotor actuation. A rigorous evaluation of the system was conducted, measuring latency across different operational modes, spatial accuracy in object grasping, and overall system responsiveness. Results indicate that the dual-mode architecture significantly enhances operational flexibility while maintaining sub-second latency thresholds acceptable for semi-structured environments.
